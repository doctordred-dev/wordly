# PWA Icons

This folder should contain the following icon sizes for PWA support:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to generate icons

1. Create a 512x512 PNG icon with your app logo
2. Use a tool like https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator to generate all sizes
3. Place the generated icons in this folder

## Recommended icon design

- Use a simple, recognizable design
- Ensure good contrast
- Use the app's primary color (#06b6d4 cyan)
- Include padding for maskable icons
